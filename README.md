# Erome-Downloader
Este é o ShurXit Downloader, um gerenciador de downloads desenvolvido em Python. Ele conta com atualização automática pelo gallery-dl e oferece monitoramento ativo para downloads automáticos ao copiar links, além de modo manual, filtro de sites e uso de cookies para resolver problemas em plataformas como Instagram e Twitter.

This is ShurXit Downloader, a download manager developed in Python. It features automatic updates via gallery-dl and offers active monitoring for automatic downloads when copying links, as well as a manual mode, website filtering, and the use of cookies to resolve issues on platforms like Instagram and Twitter.
